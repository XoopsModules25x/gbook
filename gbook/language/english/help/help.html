Help: GBook

Description


The GBook module is for managing a guestbook.


Install/uninstall


No special measures necessary, follow the standard installation process:

  • Extract the /gbook folder into the ../modules directory
  • Install the module through Admin -> System Module -> Modules

Operating instructions


  • Configure your preferences for the module (see 'Peferences').
  • Manage the GBook entries (see 'Manage Entries').
